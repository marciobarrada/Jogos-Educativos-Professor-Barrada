ABACADA: Monte as palavras conforme as figuras

Instruções:

  • O objetivo deste jogo e montar as palavras conforme os desenhos.

  • Clique na bandeira verde para iniciar o jogo, analisando as sílabas pertinentes para montar a palavra.

  • Na sequência leve o mouse sobre a letra, e clique nela para montar as palavras.

  • Para ajudar as letras estão da cor do seu desenho.

  • Após a apresentação das separação das sílabas, localize as letras, clique nelas para escrever o nome das figuras
